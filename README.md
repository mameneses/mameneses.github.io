# Matias Meneses - Professional Portfolio Website

This repository contains the source code for my personal career website, hosted with GitHub Pages.
The site showcases my professional background, selected projects, and ways to connect.

## Live Site

- URL: `https://mameneses.github.io`

## Purpose

- Present my professional profile and experience
- Highlight portfolio projects and technical work
- Provide a central place for contact and resume access

## Tech Stack

- HTML5
- CSS3
- JavaScript (jQuery + plugins)
- React
- Vite
- Yarn
- Static assets (images, icons, resume PDF)

## Project Structure

- `index.html` - Main landing page
- `css/` - Styling and animation files
- `js/` - Front-end behavior and plugin scripts
- `img/` - Images, icons, and downloadable resume
- `my_site_6_2014.html` - Legacy/archived page version

## Run Locally

Install dependencies:

```bash
yarn install
```

Start the development server:

```bash
yarn dev
```

Build for production:

```bash
yarn build
```

Preview the production build:

```bash
yarn preview
```

## Deployment

This site is designed for GitHub Pages deployment from this repository.
Any updates pushed to the published branch are reflected on the live site after GitHub Pages finishes building.

## Customization

To adapt this site for your own professional portfolio:

- Update text content in `index.html`
- Replace images and resume files in `img/`
- Adjust styles in `css/my_site.css`
- Update scripts/interactive behavior in `js/my_site.js`
