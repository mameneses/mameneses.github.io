CSS Basics Cheatsheet

By Matias Meneses

Adding Styles

External stylesheet

<head>
<link rel="stylesheet" type="text/css" href="style.css" />
</head>

Internal stylesheet

<style type="text/css">
style properties </style>

Style Syntax

Basic syntax

selector {
property: value;
}
Example:
body {
color: blue;
}

Selectors

HTML Tags=> p, body, h1-h6
Classes=> .className
IDs=> #idName
